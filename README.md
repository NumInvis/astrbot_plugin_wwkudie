# 尽孝插件 v2.0.0 - AstrBot 游戏评论生成插件

一个 AstrBot 插件，用于生成热爱游戏、把游戏公司当父亲尽孝风格的游戏深度评论文章。

## ✨ 功能特性

- 📝 自动生成尽孝风格的游戏深度评论
- 🎯 支持任意游戏，不局限于特定游戏
- 🎨 9种内置写作风格 + DIY自定义风格
- 📜 历史记录功能，查看过往生成
- ⏱️ 智能冷却和限流机制
- 🛡️ 输入安全过滤，防止 Prompt Injection
- ⚙️ 完善的配置系统
- 🔧 模块化架构，易于扩展

## 🚀 安装方法

### 方法一：通过 AstrBot 插件商店安装
1. 打开 AstrBot 管理面板
2. 进入插件商店
3. 搜索「尽孝」
4. 点击安装

### 方法二：手动安装
1. 将 `astrbot_plugin_wwkudie` 文件夹复制到 AstrBot 的 `data/plugins` 目录下
2. 重启 AstrBot
3. 插件自动加载

## 📖 使用方法

### 基础命令

```
/尽孝 游戏名 事件描述 [风格/diy 自定义风格]
```

### 使用示例

```
# 基础用法
/尽孝 鸣潮 有人黑角色西格莉卡绑男角色仇远所以不抽西格莉卡
/尽孝 原神 3.0版本须弥开放

# 使用特定风格
/尽孝 王者荣耀 新英雄上线 数据分析
/尽孝 崩坏星穹铁道 剧情太刀 激烈反问

# DIY自定义风格
/尽孝 原神 版本更新 diy 用鲁迅的口吻写
/尽孝 鸣潮 新角色 diy 模仿新闻联播播音员
```

### 其他命令

- `/尽孝风格` - 查看所有可用风格
- `/尽孝历史` - 查看您的生成历史
- `/尽孝清除` - 清除您的历史记录
- `/尽孝状态` - 查看插件和个人状态
- `/尽孝帮助` - 显示帮助信息

## 🎨 写作风格

### 内置风格

| 风格 | 描述 | 别名 |
|------|------|------|
| 默认 | 标准尽孝风格，情感真挚 | default, 正常, 普通, 标准 |
| 文言文 | 史书笔法，典雅厚重 | 古文, 文言, 古典, 史记, 春秋 |
| 赞美诗歌 | 诗意盎然，像写情书 | 诗歌, 诗意, 抒情, 文艺, poem |
| 数据分析 | 用数据说话，专业可信 | 数据, 理性, 分析, 统计, data |
| 激烈反问 | 结构化反问，层层递进 | 反问, 激烈, 质问, intense |
| 商业精英 | 互联网黑话，精英范儿 | 商业, 精英, 老板, MBA, business |
| 业内人士 | 揭秘内幕，专业接地气 | 业内, 内部, 从业者, 开发者 |
| 新闻联播 | 庄重权威的新闻播报 | 新闻, 播音, 主播, news, 央视 |
| 学术期刊 | 严谨的学术论文风格 | 学术, 论文, 科研, research |

### DIY自定义风格

使用 `diy` 关键字加上你的风格描述：

```
/尽孝 游戏 事件 diy 用鲁迅的口吻写
/尽孝 游戏 事件 diy 模仿新闻联播播音员
/尽孝 游戏 事件 diy 用 Rap 歌词的形式
```

## ⚙️ 配置说明

在 AstrBot 管理面板中可以配置：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| cooldown_seconds | 冷却时间（秒） | 30 |
| max_game_name_length | 游戏名最大长度 | 50 |
| max_event_length | 事件描述最大长度 | 500 |
| max_custom_style_length | 自定义风格最大长度 | 200 |
| max_article_length | 生成文章最大长度 | 2000 |
| enable_cooldown | 启用冷却机制 | true |
| enable_rate_limit | 启用限流机制 | true |
| rate_limit_count | 每小时最大请求次数 | 10 |
| default_style | 默认写作风格 | 默认 |
| enable_history | 启用历史记录 | true |
| max_history_per_user | 每用户最大历史记录数 | 10 |
| system_prompt | 自定义系统提示词 | - |

## 🔒 安全特性

- **Prompt Injection 防护**：自动过滤危险字符和模式
- **输入长度限制**：防止超长输入导致的问题
- **冷却机制**：防止 API 滥用
- **限流机制**：每小时请求次数限制
- **错误处理**：友好的错误信息，不暴露内部细节
